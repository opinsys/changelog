Opinsys OS releases and changes

The following changelog lists Opinsys OS (operating system) releases and changes after the previous version. In addition to the described changes, each release contains the latest security related software updates (for example, updates to the latest web browsers).

## O2219 - Taygeta {#opinsys-os-opinsys-buster-2022-05-12-121616-amd64}

- 
Opinsys OS devices can now be set to factory reset mode from the device management. On next startup 
this will make the laptop boot to a special reset mode where all the files and configuration will be securely
deleted. Setting laptop to a reset mode can be done from the devices information page and by selecting 
"Set to reset mode" from the "More Actions" menu.


- 
This whiteboard application can be installed via puavo-pkg from the device management
or by contacting our support.


- 
Support for Ukrainian language has been added to applications and UI.


- Update: Flashforge Flashprint is updated to version 5.3.3


### Known problems


### Technical changes

- Fix: Puavo-ers exam server disables suspend by default.
- Addition: Added Bluetooth test to puavo-test-hardware.
- Addition: Packages retext ja okteta.
- Addition: Puavo OS Console reports "BATTERY MISSING" for devices that do not have battery.
- Addition: Added Disk Erase option to Puavo OS Console.
- Update: Puavo-ers server supports full automation with Abitti KTP servers starting from server versions SERVER2216L.
- Update: HP Elitebook bioses can be updated to the latest version from Puavo OS Console.
- Change: Microphone and speaker test have beencompined as one audiotest in puavo-test-hardware.

## O2202 - Polaris {#opinsys-os-opinsys-buster-2022-01-10-125220-amd64}

- 
The progress of an ongoing update is now visible on shared devices, and can be found from support
applet menu. Updates can also be checked for on shared devices using the support applet.


- 
The following puavo-pkgs are available: Telegram, Konica Minolta Printer drivers, Logicsim,
and Epson EasyMP Network Projection.


- 
Shared folders on bootservers now include two new folders, which can be used e.g. for returning test
answers. Students can save their answers to "Assigment Submission Box"-folder, from which they will
be automatically collected to "Collected Assignments" folders, which is available for teachers.


- 
A startup problem of Geogebra 6 on 10th Gen Intel devices was fixed.



- 
In some rare occasions, e.g. a file with no write permissions on the users desktop could
have caused the desktop to hang. This has now been fixed.


- Fix: Fixes in Libreoffice localizations.

- Fix: Minetest-icon is again visible in Puavomenu.

- Fix: Fixed missing Swedish desktop applet translations.


### Known issues
- Due to a Google Chrome bug, dragging tabs between Chrome browser windows is not working properly.


### Technical changes

- Update: IdId-player has been updated to latest version.
- Fix: Added logging configs to prevent state partition from filling.
- Fix: Puavo-make-install-disk -tool can be given `--only-legacy-boot` parameter for
maximal support with legacy devices.
- Fix: Automatic update notifier was disabled from Nextcloud Desktop
- Change: Puavo-tftp-server on bootservers has been replaced with hook-tftp. This change enables
the booting of Bullseye-based images, and improves the tftp server performance
- Change: Savedisk -tool can be given a specific name to save as.
- Change: New `puavo.printing.cleanup_printers` -puavo-conf allows more fine-grained configuration of automatic cleaning
of printers connected to system.
- Addition: New puavo-conf `puavo.printing.airprint.enabled` allows disabling of Airprint on devices.
- Addition: A script `puavo-gather-printing-statistics` was added to boot servers, allowing to collect statistics of printing.
- Addition: Unshield-program to enable uncompression of Windows InstallShield packages.

## O2025 - Absolutno (development release) {#opinsys-os-opinsys-buster-2020-06-18-055608-amd64}

This is the first version of the upcoming Opinsys OS.
This version is still a development version, thus it may
lack some things and for example the user interface might still
undergo some small changes.  This release is however mostly
stable and is safe to use.  The devices using this release
are under normal maintainance and it is possible to update
them automatically to newer releases.

### Known issues

- Missing/faulty application icons in menu
- Selecting dark theme does not select the corresponding dark icon theme
- "Show desktop" icon can be broken depending on the  icon theme
- Kojo-programming application does not launch

### Technical changes


  - moved on to using Debian "Buster"-version
  - using Linux kernel 4.19 by default
  - most software has been updated
  - and a whole lot more...
